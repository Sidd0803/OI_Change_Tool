# OI Change Tool

```

├── src/                        # Python scripts
│   ├── template.py             # Step 1: parse Bloomberg chat log → template
│   ├── bloomberg_tickers.py    # Step 2: filter OI Change lines → bloomberg_tickers.txt
│   ├── generate_final_output.py        # Step 3a: generate final_output.txt (Element Chat)
│   └── generate_recap_input_txt.py     # Step 3b: generate recap_input.txt
├── tests/                      # Test suite
│   ├── test_bloomberg_tickers.py
│   └── test_template.py
├── data/                       # Input and output files
│   ├── dates.txt               # Monthly expiry dates (MM/DD/YY), Jun 2026 – Dec 2028
│   ├── original_input.txt      # Raw Bloomberg chat log (your input)
│   ├── template.txt            # Generated by template.py
│   ├── filtered_input.txt      # Generated by bloomberg_tickers.py (intermediate)
│   ├── bloomberg_tickers.txt   # Generated by bloomberg_tickers.py (final)
│   ├── numbers.xlsx            # OI change + volume data from Excel
│   ├── final_output.txt        # Generated by generate_final_output.py
│   └── recap_input.txt         # Generated by generate_recap_input_txt.py
└── readme.md
```

## Workflow

**1. Paste the Bloomberg chat log**

Put the previous day's chat log from the Enrique Bloomberg chat into `data/original_input.txt`.

**2. Run `template.py`**

```
python src/template.py
```

Parses the chat log and writes the trade template to `data/template.txt`. Covers most trade structures automatically (single options, call/put spreads, risk reversals — including cross-expiry). Fill in any unparsed trades manually.

**3. Run `bloomberg_tickers.py`**

```
python src/bloomberg_tickers.py
```

Filters the OI Change lines from `data/template.txt` and converts them to Bloomberg-formatted tickers in `data/bloomberg_tickers.txt`. Paste these into the color open interest spreadsheet.

**4. Paste OI change + volume data**

Take the OI change and volume columns from the spreadsheet and paste into `data/numbers.xlsx`.

**5a. Run `generate_final_output.py`**

```
python src/generate_final_output.py
```

Populates `data/final_output.txt`. Add a heading for the day and drop into Element Chat.

**5b. Run `generate_recap_input_txt.py`**

```
python src/generate_recap_input_txt.py
```

Populates `data/recap_input.txt` for the recap.

## Expiry Dates (`data/dates.txt`)

Each line is one expiry date in `MM/DD/YY` format, listed chronologically. Currently populated Jun 2026 – Dec 2028. Add or remove dates freely — the script reads whatever is present.

Supported month formats in the template:

| Format | Example | Resolves to |
|---|---|---|
| Plain month | `Jun` | First matching entry in dates.txt |
| Month + year | `Jun27` | Exact entry for Jun 2027 |
| Day + month | `15Jun` | Day 15 of first matching June |
| Month + day + ordinal | `Jun15th` | Same as above |

Month names are case-insensitive (`jun`, `JUN`, `Jun` all work).
